### Usage Examples

#### Run Unit Tests Only
```bash
go test ./internal/bot -short
```

#### Run All Tests (with Discord token)
```bash
export DISCORD_TEST_TOKEN="your_test_token"
go test ./internal/bot -v
```

#### Use Test Scripts
```bash
# Linux/macOS
./scripts/run-integration-tests.sh

# Windows
scripts\run-integration-tests.bat
```
